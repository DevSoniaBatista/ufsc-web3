# ufsc-web3

Apresentação em Next.js (App Router) sobre Web3: evolução da internet, blockchain, aplicações no mundo real e oportunidades.

## Pré-requisitos

- Node.js (recomendado 18+)
- npm

## Como executar (desenvolvimento)

No Windows (PowerShell), a partir da raiz deste repositório:

```bash
cd c:\_sonia\dev\palestra\ufsc-web3
npm install
npm run dev
```

Abra:

- http://localhost:3000

## Scripts úteis

```bash
# build de produção
npm run build

# executar build (produção)
npm run start

# lint
npm run lint
```
